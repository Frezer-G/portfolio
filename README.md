# 🌐 Frezer Getachew Portfolio

A minimalist personal portfolio built step-by-step using HTML, CSS, and JavaScript.

---

## 📌 Project Overview

This project demonstrates a clean development workflow using Git and GitHub:

* **Version 1:** Static website using HTML and CSS
* **Version 2:** JavaScript functionality added via Pull Request

---

## 🚀 Features

### ✅ Version 1 (HTML & CSS)

* Semantic HTML structure
* Clean and minimalist design
* Responsive layout
* Sections: About, Skills, Projects, Contact

### ✅ Version 2 (JavaScript)

* Interactive behavior added
* DOM manipulation
* Enhanced user experience

---

## 🔀 Git Workflow

This project follows a simple and clear Git workflow:

1. Created `version1` branch

   * Built the full UI using HTML and CSS
   * Committed step-by-step

2. Created `version2` branch

   * Added JavaScript features

3. Opened a Pull Request

   * Merged `version2` into `version1` (or main)
